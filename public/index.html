How do I update missing data?

By clicking on the edit icon () you will be presented with an edit screen where you can changes to your activities and the associated metadata.

Metadata is an important aspect of the reporting and enables the Expert program management to quantify and report on all the great activities that Experts participate in. The tool is able to gather some of this metadata on its own, in particular YouTube videos that are posted in Google+, but for many other such as blogpost views and event participants its up to the Expert to enter this data.

Here are some examples of metadata impact value for popular activities.

  • add#forumpost -> N of views [enter manually]
  • #community, #techtalk -> N of attendees [enter manually]
  • #opensourcecode -> N Star/Downloads [enter manually]
  • #video -> N of views/copies [automatic if a youtube video link]

Use the action buttons on the edit screen to discard or save your changes.
